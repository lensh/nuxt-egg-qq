# vue-qq2.0
vue-qq 2.0版本，将使用 Nuxt 和 Egg.js 对1.0版本进行重构。目前项目正在重构中。

对比1.0，会有以下优化和改进：

* 为了SSR，前端使用 Nuxt框架，提高首屏加载速度。
* 后端使用Egg.js框架。相对于Express/koa来说，Egg.js更加适合开发中大型项目。
* 使用 mongoose 操作数据库。
* jwt 实现服务端的鉴权。
* 登录密码采用sha1加盐处理以防止密码泄露。
* 防抖节流优化，提升性能。
* 注册使用邮箱注册，因为手机号注册需要付费，所以使用免费的邮箱注册。
